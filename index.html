CSS Specificity Table

Item to be styled: How it appears in HTML: How it appears in CSS: Specificity level:
All items (universal selector) n/a (only coded in stylesheet) * { style declaration } Very low.
Elements <p> (for example) p { style declaration } Low
Elements with particular classes <p class="style"> .style { style declaration } Medium
Elements with particular IDs <p id="specialStyle"> #specialStyle { style declaration } High
Inline Styles <p style="color: blue;"> n/a - no CSS styling, as it all occurs in the HTML only. Very high
Using the !important declaration You're more likely to see this bad boy in CSS. p { style declaration !important } Bigly high
